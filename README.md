## Tech stack

Built with plain HTML, CSS, and vanilla JavaScript — no build step or framework required, so it runs directly on GitHub Pages.

- `index.html` — page structure and content
- `css/style.css` — styling
- `js/main.js` — small script (footer year)
- `assets/` — project logos and downloadable résumé (PDF)

## Local preview

No build tools needed. Clone the repo and open `index.html` in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This repo is deployed via **GitHub Pages**:
1. Settings → Pages
2. Source: `main` branch, root folder
3. Site publishes at `https://sheldonalag.github.io`

## Updating content

- **Projects/experience:** edit the relevant sections directly in `index.html`.
- **Résumé:** replace `assets/sheldon-alag-resume.pdf` with an updated version (keep the same filename, or update the link in `index.html`).
- **Project logos:** add new images to `assets/img/` and reference them in the project cards.
